# Minecraft Fun! Staff Dashboard

A modern web platform for managing the Minecraft Fun! team, content pipeline, analytics, scheduling, and creative planning.

## Features

### Authentication
- Secure staff sign in system
- Staff application/signup flow
- Admin approval system for new applicants
- Role-ready structure for future expansion

### Dashboard Tools
- Creator Content Pipeline
- Episode Planner
- Analytics Dashboard
- Idea Vault
- Series Bible Lite
- Recording Checklist
- Thumbnail Planner
- Collab Tracker
- Shorts Planner
- Upload Schedule
- Asset & Resource Library
- Team Dashboard

## Planned Tech Stack

### Frontend
- Next.js
- Tailwind CSS
- TypeScript

### Backend
- Supabase Authentication
- PostgreSQL Database
- API Routes for dashboard management

### Additional Features
- Interactive analytics charts
- Calendar-style upload planner
- Responsive design for desktop and mobile
- Spreadsheet-inspired dashboard layouts
- Clean and professional UI

## Project Goals

This project is designed to help the Minecraft Fun! team organize content creation, uploads, collaborations, thumbnails, shorts, analytics, and long-term planning in one centralized platform.
